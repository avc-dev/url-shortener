# Database Migrations

Этот модуль предоставляет автоматическое применение миграций базы данных при запуске приложения.

## Особенности

- Миграции применяются автоматически при подключении к PostgreSQL
- Использует embed для встраивания SQL файлов в бинарный файл
- Интегрирован с существующей системой логов
- Безопасное применение миграций с проверкой состояния

## Использование

Миграции применяются автоматически при инициализации базы данных в `internal/app/dependencies.go`.

## Структура файлов

- `migrator.go` - основная логика применения миграций
- `schema/` - SQL файлы миграций (embed'ированные)

## Добавление новых миграций

1. Создайте пару файлов в `internal/migrations/schema/`:
   - `{version}_{name}.up.sql` - для применения миграции
   - `{version}_{name}.down.sql` - для отката миграции

2. Файлы будут автоматически включены в embed и применены при следующем запуске
